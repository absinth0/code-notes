Inheritance in CSS & the related properties

all the CSS properties can be set to these values that control inheritance of that property:

  • inherit
  • initial
  • unset

There is also a property called all that takes one of these three as input and sets all propertis of the element to that value.

  • Default link color(link do not inherit color by default. exceptional :3 )
  • Inherit the link color
  • Reset the link color

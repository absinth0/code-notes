MDN Resource

Properties to know about:

  • justify-content & align-items: Organize flex items along main axis and cross axis.
  • flex-flow: Shorthand for flex-direction and flex-wrap property.
  • flex: Shorthand for flex-grow,flex-shrink and flex-basis property. Should not use until necessary.
  • align-content: organize items that span multiple lines.
  • order: changes default order of a flex child.
  • align-self: aligns the flex element along cross axis.
